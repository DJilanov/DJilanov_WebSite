Statefull buttons Github

The button is constructed by 4 spans with code:
span class="default" data-state="default"
span class="working" data-state="working"
span class="success" data-state="success"
span class="error" data-state="error"
You can set the class with every type of class you want. We only use the data-state attribute.
When we want to change state we just type:
setState($(".btn"),"working")
Where the first argument is the object we want to change state and the second is the state we go. You can easy test it right here
$("#setOnDefault").click(function() {setState($("#select"),"default");});
$("#setOnWorking").click(function() {setState($("#select"),"working");});
$("#setOnDefault").click(function() {setState($("#select"),"success");});
$("#setOnDefault").click(function() {setState($("#select"),"error");});
